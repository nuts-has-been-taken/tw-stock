# 📚Stock Experiment

這個資料夾存放著每一個展開股市研究的主題。研究主題分為已經完成的和尚未開始的。

完成的研究主題中的 function 會經過改良，並放入 [automation_scripts](https://github.com/nuts-has-been-taken/tw-stock/tree/main/automation_scripts) 進行自動化分析其他股票。
